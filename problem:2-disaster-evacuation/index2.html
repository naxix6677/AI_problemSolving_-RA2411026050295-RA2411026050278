Disaster Evacuation Planner

1. Problem Description

Natural and man-made disasters such as floods, earthquakes, wildfires, and industrial accidents require efficient evacuation planning. This system calculates the safest and fastest routes for people to reach shelters.

  • Find shortest and safest paths
  • Avoid hazardous routes
  • Distribute evacuees efficiently
  • Adapt to dynamic conditions

2. Algorithms Used

Dijkstra’s Algorithm

Finds the shortest path from source to all nodes using non-negative weights.

A* Algorithm

Uses heuristics to speed up pathfinding.

BFS (Breadth-First Search)

Used when all edges have equal weight.

Multi-Source Shortest Path

Finds nearest shelter among multiple options.

Risk-Aware Routing

cost = distance + (risk_factor × weight)

3. Execution Steps

  1. Prepare input graph (nodes and edges)
  2. Construct adjacency list
  3. Select algorithm (Dijkstra / A*)
  4. Compute optimal path
  5. Display output
  6. Update dynamically if needed

4. Sample Input

Nodes: A, B, C, D, E (Shelter), F (Shelter)

Edges:
A-B (2)
A-C (5)
B-C (1)
B-D (4)
C-D (2)
D-E (3)
C-F (6)

Source: A
Shelters: E, F

5. Sample Output

Using Dijkstra’s Algorithm

Path: A → B → C → D → E
Total Cost: 8

Alternative:
A → C → F
Total Cost: 11

Selected Shelter: E

With Risk Consideration

Path: A → B → D → E
Total Cost: 9

Selected Shelter: E
